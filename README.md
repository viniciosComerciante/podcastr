<h2>ℹ️ How to use</h2>

    Clone this repository
    $ git clone https://github.com/viniciosComerciante/podcastr.git

    Install the dependencies
    $ yarn install
    
    Start the fake api in one terminal window
    $ yarn server

    Start the project in other one
    $ yarn dev
    
    Be happy :)
